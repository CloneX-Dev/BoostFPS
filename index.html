Мануал по оптимизации ПК для игр + Настройка NVIDIA

Данный способ поможет:

  • Уменьшить расход оперативной памяти
  • Увеличить FPS в играх

1. Настройка визуальных эффектов

ПКМ по «Этот компьютер» → Свойства → Слева «Дополнительные параметры системы» → вкладка «Визуальные эффекты» → пункт «Быстродействие». Нажмите «Параметры» и выставите настройки согласно скриншоту.

2. Отключение лишних служб

Используйте программу Easy Service Optimizer. Выберите Extreme внизу, затем вкладку Service → Check all, далее File → Apply services (или Ctrl+S) и перезагрузите компьютер.

Скачать Easy Service Optimizer

3. Отключаем телеметрию, удаляем Metro приложения

Программа DWS (Destroy Windows Spying). Настройки как на скриншоте.

Не ставьте галочку «Добавить шпионские домены в hosts» — перестанет работать Skype.
Скачать DWS

Ставим галочки как на скриншоте, переходим в «Главная» и жмем «Destroy Windows». Во вкладке «Утилиты» удаляем всё лишнее: Win10 Metro, OneDrive, отключаем UAC, Office 2016 telemetry; Windows Update — по желанию.

4. Настройки плана электропитания

Пуск → Панель управления → Электропитание → Выбираем «Высокая производительность».

5. Очищаем автозагрузку и лишние сервисы

Нажимаем сочетание клавиш WIN+R, вводим msconfig.exe. Переходим на вкладку «Службы», ставим галочку «Не отображать службы Microsoft» и снимаем галочки со всего, чем не пользуемся (кроме античитов).

6. Настройки звука Windows

ПКМ на значок динамика в трее → «Устройства воспроизведения» → выберите устройство по умолчанию → «Свойства» → вкладка «Дополнительно» → выберите 16 бит 44100 Гц. Если есть вкладка с улучшениями, отключите всё. Аналогично для записывающих устройств.

7. Всегда используйте последние версии драйверов

NVIDIA Driver AMD Driver Realtek HD Drivers

8. Настройки в панели управления NVIDIA

Настройте всё в точности как на скриншотах. Это самый важный пункт для увеличения FPS. В зависимости от конфигурации ПК может дать прирост от 30 до 100 кадров.

9. Quick CPU

Установите план электропитания «Высокая производительность» и выкрутите все ползунки на 100%. Программа не позволит простаивать ядрам процессора и повысит производительность в играх, чувствительных к CPU.

Заключение

Также обязательны регулярные процедуры:

  • Физическая чистка ПК (каждые 4–6 месяцев)
  • Проверка системы на вирусы (еженедельно)

Если вы выполнили все пункты, ваш FPS обязательно вырастет, и играть станет приятнее. Спасибо за покупку!
